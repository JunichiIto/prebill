## PreBillとは

PreBillというサービスは、 定額制サービスを多数利用していると利用料金や支払日の管理が煩雑になるという問題を解決するための、サブスクリプションサービス利用者に向けた支出管理+リマインダーサービスです。

利用しているサービスの料金や更新日を一覧でき、Excel等で管理するのとは違い、任意の日時に更新日が近いことを知らせるメールを受け取ることができます。

## インストール

```
$ bin/setup
$ rails server
```

## テスト

```
$ .bin/test
```
